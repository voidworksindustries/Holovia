Patch Notes

Latest update notes for Holovia.

Version 1.8.0.1

Latest

Holovia 1.8.0.1 focuses on polish, Game Center improvements, balancing, startup quality-of-life features, and multiple gameplay fixes following the 1.8 release.

Improvements & Changes

  • Startup tips have been added
  • A version number is now displayed in the settings menu
  • Share links now correctly use the App Store link on iOS
  • Firebase SDK has now been added to iOS
  • Climb mode early-game balancing has been adjusted slightly for new players
  • Power-ups can no longer be missed because of badly timed collection moments
  • Reinstalling the game will now properly restore previous Game Center scores
  • Rush and Shift Game Center connections now work correctly

Bug Fixes

  • “Close to high score” skin feedback now only appears when the score is actually close to the required high score
  • Yellow skin now displays correctly

Version 1.8

Holovia 1.8 brings iOS support, Game Center integration, new environments (Daylight, Sunset, Night), a full performance overhaul, smoother controls, a new streak system, and major gameplay improvements. Plus dozens of bug fixes and visual upgrades.

New Features

  • iOS support — Holovia is now available on the App Store
  • Game Center integration — leaderboards and future challenges enabled
  • New environments and styles: Daylight, Sunset, and Night, including new terrain variations
  • Mission rewards system — collect rewards directly from the mission menu
  • Unique player ID system — enables skin ownership and reward tracking
  • Animated UI buttons — all buttons now respond visually to input
  • New loading screen
  • Credits screen added
  • Birds added for ambient world detail
  • Firebase integration for improved feedback and analytics
  • Bug reporting system — report issues directly from the settings menu
  • Notifications system added
  • New streak system — build streaks for bonus points and rewards
  • Cube personality added and visible in the menu

Improvements & Changes

  • Adaptive performance system automatically adjusts performance based on device
  • Manual performance settings added in the redesigned settings menu
  • Major performance overhaul, especially for low-end devices
  • Input smoothing added to reduce stutter and improve responsiveness
  • Improved haptic feedback on iOS
  • Wall generator can now spawn multiple walls at once
  • Visual enhancements including deeper gridline colors
  • Twisting walls now change colour during rotation
  • New and improved high-speed effects
  • Pause button moved to a new location
  • Power-up UI moved behind the pause menu
  • 2x indicator moved behind the pause menu
  • Mission text size adjusted
  • All walls and speeds have been rebalanced
  • Distance between grid switches has been reduced
  • Classic mode no longer unlocks new gamemodes, allowing it to remain a practice-focused mode
  • Some skins now require a high score, streak, or games played to unlock
  • Walls are now slightly slower directly after a revive, giving players a fairer chance to recover
  • Music now continues correctly after a restart
  • Early-run flavour text has been added
  • A larger reward has been added for completing the streak
  • Some cubes now have faces for added personality

Bug Fixes

  • Shooter pieces no longer shoot through other walls
  • You can no longer shoot through high-speed moving walls
  • Dash through 10 red walls now works properly
  • Side-shifting walls no longer move after starting a dash
  • Shifting walls now always show a green indicator hole
  • Rotating walls no longer show a player highlight
  • Power-ups no longer get collected unless the player actually catches them
  • You can no longer dash through a twisting wall while it is twisting
  • Twisting walls no longer kill the player during the tutorial
  • Fire has been removed from twisting 5x5 walls
  • Shooter pieces now only fire when in front of the player, unless they are self-shooting wall pieces
